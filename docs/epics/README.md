## Epic Lifecycle

```
backlog → on-progress → coding → review → testing → deploying-dev → ready-for-qa → done
                                                                              ↓
                                                                           blocked
```

## Conventions

- Each epic lives in its own `EPIC-XXX-slug.md` file.
- Status is updated in the frontmatter as work progresses.
- Every code decision is logged in the epic's **Automation Log** section.
- Tasks reference Engineering Task IDs from `docs/product/ENGINEERING-TASKS.md`.
