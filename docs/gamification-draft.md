# BRAG Gamification Draft

This document captures initial gamification ideas. Final point values will be defined later by the committee.

## Point Categories

### TYFCB

Thank You For Closed Business.

Possible scoring options:

- Fixed point per submitted TYFCB.
- Point based on transaction amount.
- Tiered amount range.
- Bonus if submitted during booster day.

### Visitor Brought

Member brings a visitor to BNI Grow.

Possible scoring options:

- Point when visitor is submitted.
- Higher point when visitor confirms attendance.
- Higher point when visitor actually attends.
- Bonus if visitor matches target business classification.

### Referral

Member gives qualified referral.

Possible scoring options:

- Fixed point per referral.
- Bonus if referral converts.
- Bonus if referral category is strategic.

## Booster Ideas

### TYFCB Day

TYFCB submitted on selected day gets extra points.

### Business Classification Hunt

Visitor from targeted business classification gets bonus.

### Referral Sprint

Referral submissions during active sprint window get bonus.

### Comeback Bonus

Member who reactivates after inactive weeks gets bonus.

### Group Streak

Group gets bonus if all members submit at least one contribution in a week.
